# Các folder - folder đó chứa gì?
*  data - chứa toàn bộ là 2748 hình ảnh, là dataset của đề tài
*  history - chứa các file history, lưu các giá trị loss, accuracy trong quá trình train

*  models - chứa các model được lưu
